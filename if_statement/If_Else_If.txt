Test Data :
    Input:
        Input the amount: 375
    Expected Output:
        There are:
        3 Note(s) of 100.00
        1 Note(s) of 50.00
        1 Note(s) of 20.00
        0 Note(s) of 10.00
        1 Note(s) of 5.00
        0 Note(s) of 2.00
        0 Note(s) of 1.00
